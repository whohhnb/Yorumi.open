# Yorumi 🎬

**追番神器 · 动漫客户端**

支持 Android / Windows，为动漫爱好者打造的高颜值播放体验。

---

## ✨ 特色功能

### 🎞 追番追剧
- 首页推荐、最新上架、继续观看
- 番剧 / 电影分类浏览，一键收藏追番
- 集数切换，自动连播

### 🎥 播放器
- **mpv 内核驱动**，硬解加速 / 渲染器自由切换
- **Anime4K 超分辨率** — 低清变高清，老旧片源焕发新生
- 播放速度 0.25× ~ 4.0×，补番快进、细节慢放随心调
- Android 手势滑动调节亮度和音量
- 桌面端鼠标滚轮调音量、键盘全快捷键
- **低内存模式** — 老设备也能流畅播放
- 断点续播、跳过片头片尾

### ❤ 收藏管理
- 番剧库 / 电影库分开管理
- 红心标记，一键取消

### 🎨 个性化
- 主题色自由定制
- 动态配色（跟随壁纸）
- OLED 纯黑深色模式

### 📥 应用内更新
- 自动检测新版本
- 下载进度实时显示
- 下载完成一键安装（Android APK / Windows MSIX）

| 按键 | 功能 |

## ❓ 常见问题

**Q：连接不上服务器？**  
请检查网络连接。

**Q：Windows 版卸载不干净？**  
MSIX 包使用固定证书签名，所有数据写入应用专用目录，通过「设置 → 应用 → Yorumi → 卸载」即可完整清除，不留残留。

**Q：低内存模式有什么用？**  
将视频缓冲从 1500MB 限制为 15MB，适合内存较小的设备，避免播放卡顿或闪退。

**Q：如何反馈问题？**  
请访问 GitHub Issues 提交反馈。
